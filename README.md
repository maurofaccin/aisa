# AISA

Auto-Information State Aggregation

This is a python module aimed at optimizing state aggregation based on the maximization of Auto-Information on dynamical processes evolving on networks.

## Requirements

- `numpy` and `scipy`
- `networkx`
- `tqdm` (optional)
